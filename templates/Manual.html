How to Use the Tuya Energy Dashboard

This site helps monitor, visualize, and control your IoT-based smart socket with real-time and historical energy profiling.

1. Overview

View live power readings, control the device state (ON/OFF), and analyze historical data by day or custom range.

2. Navigation & Controls

  • Mode: Switch between Live, Load Day, Load Range.
  • Date Picker: Available for Load Day and Load Range.
  • Apply: Fetch and update data for selected time window.
  • Download CSV: Export stored readings.
  • How to Use: Opens this page.

3. Live Mode

  • Toggle Power: Click ON/OFF buttons to control the socket.
  • Status: Shows if socket is ON or OFF in real time.
  • Live Readings: Power (W), Voltage (V), Current (A) update every 5s.
  • Todayâ€™s Energy: Displays total kWh and cost.

4. Load Day Mode

  1. Select Load Day from the mode dropdown.
  2. Pick a specific date.
  3. Click Apply.
  4. Top of chart shows total kWh and estimated cost.
  5. Charts show historical values of W, V, A.

5. Load Range Mode

  1. Select Load Range.
  2. Specify start and end date-times.
  3. Click Apply.
  4. Top of chart shows total kWh and estimated cost.
  5. Graphs show values across the interval.

6. Charts Explanation

  • Line Charts: Used for Load Day and Range; X = time, Y = measurement.
  • Bar Chart: In Live mode, shows hourly energy profile.
  • Hover over points for exact values.

7. Export Data

Click the Download CSV button to export readings in the format:

ts,power,current,voltage,kwh

8. Troubleshooting

  • Zero Readings: Make sure your socket is connected and functional.
  • Cost not correct? Email the admin : 2022-1-60-013@std.ewbubd.edu.
  • Device empty but not getting reading? Sometimes it takes a while to load data, please be patient.
